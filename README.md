# Au P'tit Destock

Project carried out during my professional training internship as a web and mobile developer; this project corresponds to a food clearance store, hygiene, and much more.

## Run Locally

Clone the project

```bash
  git clone https://github.com/maximegenetelli/AuPtitDestock.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  composer install
```

Start the server

```bash
  symfony server:start
```
